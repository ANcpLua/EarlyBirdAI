# Gods Evolve

Use this command after `/gods:review` when the selected team needs improvement.

Read `results/god_team.json` and `results/god_team_scores.csv`. Improve the input skill text, workflow axes, or candidate
god corpus only when the evidence shows missing coverage, excessive overlap, weak name usability, or a misleading role
assignment.

Do not promote the initial hypothesis to a fixed pantheon.
